"Debloating Redmi Note 10"

Published: 2021-08-20
Last Modifed: 2021-08-21

  • android

Assuming you are inside the adb shell, running the below commands would remove all the unnecessary apps from Redmi Note 10. This does NOT require you to be root. However, as expected, the .apk of these apps would still be available inside /data/app folder.

pm uninstall --user 0 com.miui.screenrecorder                     # Screen recorder
pm uninstall --user 0 com.google.android.apps.subscriptions.red   # Google One
pm uninstall --user 0 com.xiaomi.miplay_client                    # MiPlayClient
pm uninstall --user 0 com.miui.qr
